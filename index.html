TSP Genetic Algorithm Visualizer

This is a visualizer for the Genetic Algorithm applied to the Traveling Salesman Problem (TSP). It allows you to interactively simulate how the algorithm evolves through generations to find the shortest route that visits all given cities.


The population size and mutation rate can be adjusted to see how they affect the convergence of the algorithm. The genetic algorithm works by selecting two parent routes, crossover to create offspring, and applying mutation to introduce diversity.
